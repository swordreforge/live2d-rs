## Environment Requirements

- Python 3.7+
- live2d-python package
- OpenGL related dependencies

## Quick Start

1. First, ensure you have installed the necessary dependencies:

```bash
pip install -r requirements.txt
```

This includes following the steps in the main README.md to install live2d-py.

2. Copy required files from the example code:
- `resources.py`
- `model folder` (configure according to your specific situation)

3. Ensure your project structure looks like this:

```
your_project/
│
├── main.py
├── resources.py
└── Resources
```

4. Copy the example code and replace main.py, then run it.
